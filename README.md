# 蒼 — AOI WELLNESS HOUSE

`fitness-gym` シリーズ 4 番目、**「整える」ペルソナ**のための架空ウェルネスハウス。サウナ・水風呂・外気浴・ピラティス・瞑想を主軸にした retreat 型ジムを、日本の「ととのう」文化と Aman Spa 的な静寂表現の交点に置いてデザインしました。

## 主題

- **鍛える / 痩せるではなく、整える**。煽り CTA・Before/After・派手な配色を排除
- **配色**: 墨 `#1c1a17` × 苔 `#5a6b4a` × 木目 `#8c6a4a` × 生成り `#f1ece2` × 障子 `#e8e2d3` × 石 `#b7b1a4`
- **書体**: Shippori Mincho B1（見出し・縦組み）+ Noto Sans JP Light（本文）+ Inter（ラテン）+ Klee One（引用・キャプションのみ）
- **演出**: 湯気の slow fade in（4 秒）、縦組み明朝、「→」によるととのうフロー図、罫線一本の支配

## セクション

1. Hero — 縦組み「ととのう」+ 湯気フェードイン
2. 五つの間（施設紹介）— サウナ／水風呂／外気浴／ピラティス／瞑想
3. ととのうまでの順序 — 入浴 → サウナ → 水風呂 → 外気浴 → 休
4. 週間クラス表
5. ご利用案内（料金）— ドロップイン / 月会員 / 個室会員
6. 哲学（「整えるとは、引いていく時間である」）
7. 案内する人 — 整え手・呼吸案内人
8. アクセス・営業・持ち物
9. 招き CTA（ご予約 / 見学のご案内）
